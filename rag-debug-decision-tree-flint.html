RAG Debugging Decision Tree

Do not start with the LLM. Start with the evidence the LLM received.

Start at the source If the answer is wrong, first prove the correct fact exists in the documents.
Then inspect retrieval Check parsing, chunking, embeddings, index freshness, filters, ranking, and reranking.
Blame the model last If the right chunks reached the prompt, then improve grounding, citations, and model choice.
